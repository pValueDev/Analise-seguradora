# Analise-seguradora
## Descrição do Projeto

Esta análise foi realizada a partir de uma base de dados de sinistros de seguros (Insurance Claims). O principal objetivo do estudo foi explorar os dados, identificar padrões e extrair informações relevantes relacionadas aos acidentes registrados.

Durante a análise, foram investigados indicadores como a frequência de acidentes, a distribuição da severidade dos acidentes e a idade média dos envolvidos em cada nível de severidade. Além disso, foram aplicadas técnicas estatísticas, incluindo modelos de regressão, com o objetivo de avaliar possíveis relações entre duas ou mais variáveis presentes na base de dados.

Este projeto foi desenvolvido com fins educacionais, buscando aprimorar conhecimentos em análise de dados, estatística aplicada e programação em R, além de contribuir para a construção do meu portfólio profissional.

## Project Description

This analysis was conducted using an insurance claims dataset. The main objective of the study was to explore the data, identify patterns, and extract meaningful insights related to recorded accidents.

The analysis investigated indicators such as accident frequency, accident severity distribution, and the average age of individuals involved in each severity category. In addition, statistical techniques, including regression analysis, were applied to examine potential relationships between two or more variables in the dataset.

This project was developed for educational purposes, aiming to strengthen my skills in data analysis, applied statistics, and R programming, while also contributing to the development of my professional portfolio.
