# Analise-seguradora

## Project Description

This analysis was conducted using an insurance claims dataset. The main objective of the study was to explore the data, identify patterns, and extract meaningful insights related to recorded accidents.

The analysis investigated indicators such as accident frequency, accident severity distribution, and the average age of individuals involved in each severity category. In addition, statistical techniques, including regression analysis, were applied to examine potential relationships between two or more variables in the dataset.

This project was developed for educational purposes, aiming to strengthen my skills in data analysis, applied statistics, and R programming, while also contributing to the development of my professional portfolio.
